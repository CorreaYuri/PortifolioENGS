# Identidade Visual

Referencia principal: `idvisual.png`.

## Marca

- Nome: CS Command.
- Assinatura: Command. Coordinate. Deliver success.
- Conceito: central de comando, coordenação, Customer Success, ação e impacto.

## Cores

| Token | Hex | Uso |
| --- | --- | --- |
| Navy | `#0B132B` | Marca, texto principal, fundos institucionais |
| Primary Blue | `#2563EB` | Ações principais, navegação ativa, gráficos |
| Teal | `#14B8A6` | Sucesso, saúde, coordenação, confirmações |
| Ice | `#F1F5F9` | Fundos e areas neutras |
| Slate | `#64748B` | Texto secundario, legendas, icones neutros |

## Tipografia

- Familia: Inter.
- Pesos previstos: Regular, Medium, Semibold, Bold.
- Uso: titulos densos, labels curtos, tabelas escaneaveis e textos operacionais.

## UI

- Cards brancos com borda discreta.
- Raio de 8px para cards, formulários, botoes e tabelas.
- Botoes primarios azuis.
- Estados positivos em teal.
- Alertas em laranja/vermelho.
- Layout de produto operacional, sem linguagem de landing page.
- Sidebar compacta por padrao, expansivel no hover, mantendo foco no conteúdo.
- Topbar translucidA com busca, contexto do cliente e usuário.
- Hierarquia visual contida: sombras leves, cards densos, tipografia menor e pesos fortes apenas para dados-chave.
- Componentes devem parecer operacionais e refinados, evitando blocos grandes demais ou contraste pesado sem funcao.

## Direcao de interface

A interface deve parecer um centro de operação: limpa, densa, escaneável e pronta para uso diário. O foco visual deve estár em dados, status, prioridades, responsáveis e próximas acoes.
