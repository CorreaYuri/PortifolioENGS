# Checklist E2E dos Fluxos Criticos

## Comercial -> Proposta -> Projeto

1. criar lead
2. criar proposta
3. aprovar proposta
4. abrir projeto a partir da proposta
5. validar reflexo em CRM e Projetos

## RH Documento -> Assinatura

1. RH envia documento com assinatura obrigatoria
2. colaborador assina no Meu RH
3. validar snapshot assinado
4. validar auditoria de documento RH
5. validar baixa de notificacao pendente

## Financeiro/Fiscal Basico

1. registrar recebivel
2. registrar despesa
3. validar fluxo de caixa
4. registrar nota no fiscal
5. validar rastreabilidade minima de dados

## Integracoes

1. executar health-check por conector
2. executar health-check geral
3. validar log em `IntegrationLog`
